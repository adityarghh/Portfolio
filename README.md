# Personal Portfolio — Aditya Raj

A modern, interactive personal portfolio built using React + Vite and deployed on Vercel.

## 🚀 Features
- Interactive UI with smooth animations
- Resume preview + download integration
- Project showcase (AI, Web, Distributed Systems)
- Responsive and minimal design

## 🛠 Tech Stack
- React
- Vite
- TypeScript
- TailwindCSS
- Framer Motion
- Lucide Icons

## 🔗 Live Site
https://portfolio-pied-tau-95.vercel.app/

## 📄 Resume
Available directly on the site via embedded preview and download.
